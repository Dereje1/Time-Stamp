Timestamp Microservice

User Stories

  1. I can pass a string as a parameter, and it will check to see whether that string contains either a unix timestamp (ms from epoch) or a natural language date (example: January 1, 2016).
  2. If it does, it returns both the Unix timestamp and the natural language form of that date.
  3. If it does not contain a date or Unix timestamp, it returns null for those properties.

Example Input


OR paste query directly into browser in the format below:

https://timestamp-micro-serv.glitch.me/september%2022,2017

https://timestamp-micro-serv.glitch.me/1506052800000

Example Output

{"unix":"1506052800000","natural":"Fri Sep 22 2017"}
